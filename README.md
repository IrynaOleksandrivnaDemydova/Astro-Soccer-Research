# Футбольные звезды: типовые показатели вратарей
Исследование посвящено поиску и статистическому подтверждению типичных астрологических констелляций в натальных картах профессиональных футбольных вратарей.

Полный текст исследования опубликован по ссылке: [Футбольные звезды: типовые показатели вратарей](https://dobrosotskaya.online/blog/vratari)

## Цель исследования
Определить, какие астрологические факторы наиболее часто встречаются у вратарей и имеют ли они статистическую значимость по сравнению с контрольной группой и с группой полевых игроков.

## Данные
* **Источник:** Астрологическая база данных Луис Роден (Astro-Databank).
* **Критерии отбора:** Только карты с рейтингом **AA** (свидетельство о рождении) и **A** (запись по памяти).
* **Особенности:** В выборку включены только игроки с точно известным временем рождения.
* **Файлы данных:** `birth_data.csv` содержит сырые данные для расчета.

## Стек технологий
Проект реализован на **Python** с использованием следующих библиотек:
* `swisseph` (pyswisseph) - высокоточные расчеты эфемерид.
* `geopy` & `timezonefinder` - работа с географическими координатами и временными зонами.
* `pandas` - обработка и структурирование данных.
* `scipy.stats (fisher_exact)` - расчет точного теста Фишера для проверки статистической значимости гипотез.
* `itertools` - генерация комбинаций аспектных пар.

## Структура проекта
* `AstroResearch_Soccer_Goalkeepers` - основной файл с кодом исследования.
* `sefstars.txt` - файл для корректного расчета положений неподвижных звезд.
* `birth_data.csv` - датасет с данными рождения, необходим для расчетов и на его основе созданы файлы ниже.
* `results.csv` - итоговые расчеты астрологических показателей.
* `statistic_check_single_parameter.csv` - результаты проверки отдельных астрологических параметров на статистическую значимость.
* `frequency_few_aspect_check.csv` - анализ частотности связок из нескольких аспектов.
* `goalkeeper_outfield_analysis_results.csv` - расчет статистически значимых комбинаций для вратарей по сравнению с полевыми игроками. 
* `top_30_goalkeeper_signatures.csv` - наиболее характерные аспекты вратарей.

## Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install pyswisseph pandas geopy timezonefinder scipy`.
3. Запустите Jupyter Notebook для воспроизведения расчетов.

## Примечание
1. Убедитесь, что файлы `birth_data.csv` и `sefstars.txt` находятся в той же папке, что и ноутбук `AstroResearch_Soccer_Goalkeepers`, для корректного запуска расчетов.
